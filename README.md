# TaskManager API

## Описание проекта
TaskManager API — это backend-сервис для управления задачами сотрудников.  
Проект позволяет создавать пользователей, назначать им задачи, отслеживать статус выполнения и получать простую аналитику.  

Сервис реализован на FastAPI с использованием PostgreSQL и Redis, что обеспечивает быстрый отклик и удобную структуру данных.

---

## Технологии и стек
- **FastAPI** — основной фреймворк для API
- **SQLAlchemy** — ORM для работы с PostgreSQL
- **Pydantic** — схемы для валидации и сериализации данных
- **PostgreSQL** — реляционная база данных
- **Docker** — контейнеризация сервиса
- **Git** — контроль версий и управление репозиторием

---
- **Микросервисная логика**: каждый модуль отвечает за отдельную сущность (пользователи, задачи, аналитика).  
- **Dependency Injection** используется для работы с сессиями БД и упрощает тестирование.

---

## Запуск проекта
1. Клонировать репозиторий:
    ```bash
    git clone <https://github.com/nyorra/Task_manager_API.git>
   
2. Установить зависимости:
   ```bash 
   pip install -r requirements.txt
   
3. Запуск сервера FastAPI:
    ```bash
    uvicorn main:app --reload

4. Запуск Docker сервера:  
   **Убедись, что у тебя установлен [Docker](https://www.docker.com/ )**

   Запуск всех сервисов:
   
   ```bash
   docker compose up
   ```
